# 농림축산식품 공공데이터 포털 대시보드

농림축산식품 공공데이터 포털의 개방데이터 목록 API를 활용한 데이터 조회 및 시각화 웹 애플리케이션입니다.

## 기능

- 📋 **개방데이터 목록 조회**: 농림축산식품 공공데이터 포털의 개방데이터 목록 조회
- 🔍 **필터링 기능**: 데이터셋명, 제공기관으로 필터링
- 📊 **상세 정보 표시**: 각 데이터셋의 상세 정보를 카드 형태로 표시
- 🎨 **반응형 디자인**: 모바일 및 데스크톱 환경 지원

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

```

### CORS 문제 해결

공공데이터 포털 API는 CORS 정책으로 인해 브라우저에서 직접 호출이 제한될 수 있습니다. 

1. **프록시 사용**: `vite.config.js`에 프록시 설정이 포함되어 있습니다.
2. **서버 사이드**: 백엔드 서버를 통해 API를 호출하는 방법도 있습니다.

실제 API 엔드포인트가 다른 경우 `src/App.jsx` 파일의 `BASE_URL`을 수정하세요.
